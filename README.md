# HeaderPwnBurpEXT - Burp Suite Extension

HeaderPwnBurpEXT extension for Burp.

## 🚀 Features
- A port of another project on Github [HeaderPWN](https://github.com/devanshbatham/headerpwn) to workin burp as a single workflow.

## 📋 Requirements

- **Burp Suite Professional** with or without AI features enabled
- **Montoya API 2025.8** or later
- **Java 17+** runtime environment
- 2 files (1 for user agents, 1 for headers you want to test)
  
## 🛠 Installation

### Method 1: Load the py script
1. Download the repo
2. Open Burp Suite Professional
3. Go to **Extensions** → **Installed** → **Add**
4. Select **Python** as extension type
5. Choose the py file and click **Next**

## 🎯 Usage
- Tool runs when you tell it to from its GUI
